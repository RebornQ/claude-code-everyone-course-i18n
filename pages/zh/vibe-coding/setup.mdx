## 什么是氛围编程（Vibecoding）？

氛围编程指的是：你通过描述你想要的结果来构建软件，而不是自己写代码。

这个词因 Andrej Karpathy 而流行——“完全沉浸在 vibes 里”。你说清楚结果，AI 负责实现细节。

关键技能不是编码，而是：
- 把目标描述清楚
- 给出高质量反馈
- 判断什么才是“对的”

你做的是编排，而不是实现。

## PM/工程师协作关系

这就是非技术创始人和开发者的真实协作方式：

- **你的工作：** 明确你要什么，并给出清晰反馈
- **对方的工作：** 想办法把它做出来

在这个模块里，你扮演 PM，Claude 扮演工程师。

你描述。Claude 构建。你审核。Claude 迭代。

这是一项可迁移的职业技能：和设计师、开发者、代理公司，甚至任何 AI 工具协作都适用。

## 构建循环（Build Loop）

所有项目都遵循同一套循环：

1. **规划（Plan）** → 把要做什么说清楚
2. **构建（Build）** → 让 AI 把它做出来
3. **迭代（Iterate）** → 不断修改到满意
4. **保存（Save）** → 用 GitHub 备份
5. **上线（Go Live）** → 部署到互联网

这个循环适用于任何项目：落地页、工具、应用、原型都一样。

## 什么是框架（Framework）？

框架是一个预搭好的“地基”，让你不用从零开始。

你可以把它想成房子的骨架：先有框架，再装墙和刷漆。它处理通用问题，让你专注在你的应用本身。

我们使用 **Next.js** 这个框架：它很流行，适合做网站，并且与 Vercel（我们之后要用来部署的服务）配合很好。

你不需要深入理解它，只要知道它存在即可。

## 什么是 JavaScript？

JavaScript 是最常见的 Web 编程语言，运行在浏览器里，让网页变得可交互。

你不需要学它——Claude 会替你写。
