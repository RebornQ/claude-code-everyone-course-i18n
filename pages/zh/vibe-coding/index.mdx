## 模块 2：氛围编程（Vibe Coding）

- **预计用时：** 1-2 小时

![Vibe Coding 101 - From idea to app with Claude Code](/images/vibe-coding-banner.png)

你已经学会了“与文件一起工作”。现在来学“创建文件”。

在这个模块里，你会从零构建一个真实的 Web App，并把它部署到互联网上——不需要任何编程经验。你描述你想要什么，Claude 负责实现，然后你通过不断迭代把它打磨到满意为止。

## 你会构建什么？

一个性格测试小应用，包含：
- 多道题（可自定义逻辑）
- 基于答案计算性格类型结果
- 一个任何人都能访问的线上 URL

同样的方法也适用于落地页、工具、仪表盘、原型——任何 Web 形态的东西。

## 你会学到什么？

| 课程 | 内容 |
|--------|----------------|
| **2.1：准备** | 氛围编程的心法与项目文件夹初始化 |
| **2.2：规划** | 通过互动式访谈把需求说清楚 |
| **2.3：构建与迭代** | 脚手架、开发、用截图做反馈不断优化 |

### 下载课程材料

**如何打开终端：**
- **Mac：** 按 `Cmd + Space`，输入 “Terminal”，回车
- **Windows：** 按 Windows 键，输入 “PowerShell”，回车

### 方式 1：一条命令（推荐）

**Mac/Linux：** 把下面整条命令复制并粘贴到终端：
```bash
cd ~/Documents && \
curl -L https://github.com/carlvellotti/claude-code-everyone-course/releases/download/v2.0.0/complete-course.zip -o cc4e-course.zip && \
unzip -o cc4e-course.zip && \
cd cc4e-course && \
claude
```

**Windows：** 使用方式 2 或 3。

### 方式 2：让 Claude 来做

1. 打开终端并输入 `claude`
2. 对 Claude 说：“Download the course materials from https://github.com/carlvellotti/claude-code-everyone-course/releases/download/v2.0.0/complete-course.zip and extract it to a folder called cc4e-course in my Documents folder”
