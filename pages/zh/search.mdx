# 搜索

使用下方搜索框在所有课程页面中查找内容。搜索由 Pagefind 提供，完全在浏览器端运行（不需要服务器）。

<link rel="stylesheet" href="/pagefind/pagefind-ui.css" />
<div id="search"></div>
<script src="/pagefind/pagefind-ui.js"></script>
<script dangerouslySetInnerHTML={{__html: `
  if (typeof window !== "undefined") {
    window.addEventListener("DOMContentLoaded", () => {
      new PagefindUI({
        element: "#search",
        showImages: false,
        baseUrl: "/",
        showSubResults: true
      })
    })
  }
`}} />

## 搜索小技巧

- **精确短语：** 用引号包起来，例如 `project memory`
- **多个关键词：** 搜索 `PRD template` 可以找到同时包含两者的页面
- **结果定位：** 结果会显示命中内容来自哪个章节

## 找不到想要的内容？

如果你没找到想要的内容：
- 尝试换关键词或同义词
- 从侧边栏导航按模块浏览
- 先从 [快速开始](/zh/getting-started/introduction) 入门
