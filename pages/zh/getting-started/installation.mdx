## 0.1：安装

- **预计用时：** 15 分钟

## 前置条件

- **Claude Pro 或 Max 订阅**（不是免费档）
  - 没有的话可在 [claude.ai](https://claude.ai) 注册
  - Pro 为 $20/月
- Mac（macOS 10.15+）、Windows 10+ 或 Linux（Ubuntu 20.04+）
- 4GB+ 内存
- 网络连接
- **不需要任何编程工具！**

## 如何打开终端

**Mac：**
1. 按 `Cmd + Space` 打开 Spotlight
2. 输入 “Terminal”
3. 回车

**Windows：**
1. 按 Windows 键
2. 输入 “PowerShell”
3. 回车

你会看到一个带闪烁光标的窗口——这就是输入命令的地方。

## 安装 Claude Code

复制下面的命令，粘贴到终端里并回车。

**Mac/Linux：**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Mac（Homebrew）：**
```bash
brew install --cask claude-code
```

**Windows PowerShell：**
```powershell
irm https://claude.ai/install.ps1 | iex
```

运行完成后，关闭并重新打开终端。

## 验证安装

```bash
claude --version
```

你应该会看到一个版本号。如果出现 “command not found”，请关闭终端重新打开再试一次。

## 登录认证

1. 输入：`claude`
2. 选择 **“Claude account with subscription”**（不是 Anthropic Console）
3. 完成浏览器登录
4. 选择你偏好的文本风格
5. 你会看到 Claude Code 的欢迎界面

## 常见问题排查

### command not found: claude
- 完全关闭终端后重新打开
- Mac：运行 `source ~/.zshrc` 或 `source ~/.bashrc`
- Windows：重启 PowerShell

### Subscription required
- 在 [claude.ai](https://claude.ai) 确认订阅处于激活状态
- 退出登录再重新登录

### Permission denied
- Mac：安装脚本会自动处理权限
- Windows：以管理员身份运行 PowerShell

### 运行 `/doctor` 做诊断
进入 Claude Code 后输入 `/doctor`，它会显示哪些功能正常、哪些不正常。

## 命令速查

| 命令 | 说明 |
|---------|-------------|
| `claude` | 启动 Claude Code |
| `claude --version` | 查看 Claude Code 版本 |
| `claude doctor` | 运行系统诊断（在终端中） |
| `/doctor` | 运行健康检查（在 Claude Code 内） |
| `/help` | 查看可用命令（在 Claude Code 内） |
