## 0.0：Claude Code 简介

## 什么是 Claude Code？

Claude Code 是一个直接在你电脑上工作的 AI 助手：

- **不只是写代码** - 它是通用型助手，背后是 Claude（例如 Opus 4.5）
- **能读文件、分析数据、生成文档、执行命令**
- **生活在终端里** - 但你只需要像聊天一样对话
- **你唯一需要记住的命令：** `claude`

## 这门课适合谁？

- 想把 AI 用在真实工作里的**非技术背景**人群
- **不需要编程经验**
- **不需要终端经验**
- 只要愿意尝试与保持好奇心

## 课程结构

| 模块 | 重点 | 时长 |
|------|------|------|
| **模块 1：基础** | 文件操作、agents、记忆 | ~3 小时 |
| **模块 2：氛围编程（Vibe Coding）** | 构建一个真实 Web App（即将上线） | ~2-3 小时 |
| **未来模块** | MCP 集成、浏览器自动化 | TBD |

## 前置条件

- Claude Pro 或 Max 订阅（在 [claude.ai](https://claude.ai) $20/月）
- Mac / Windows / Linux 电脑
- 模块 1 约 3 小时

## Claude Code vs ChatGPT

| 能力 | ChatGPT | Claude Code |
|---------|---------|-------------|
| **文件访问** | 无 | 可读写任意文件 |
| **记忆** | 会话结束就消失 | 持久（CLAUDE.md） |
| **执行** | 建议你怎么做 | 真的帮你做 |
| **多文件处理** | 一次一个 | 一次几十个 |
| **Agents** | 单实例 | 并行处理 |

**关键差异：** Claude Code 不只是给建议——它会和你一起把事做完。

## 思维转变

> 从“我得学一堆命令”到“我只要这个结果”

- 你学的是**编排**，不是写代码
- 你描述目标，Claude 负责实现路径
- 终端只是你和它对话的地方
