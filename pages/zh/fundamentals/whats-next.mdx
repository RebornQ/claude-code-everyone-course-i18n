| 工具 | 作用 |
|------|--------------|
| **文件探索** | 快速进入陌生文件结构 |
| **单文件 → 提取** | 从混乱中提炼结构 |
| **文件夹 → 综合** | 在大量文件中找规律 |
| **套模板** | 控制输出格式 |

## 预览：自定义 Skills

就像 `/start-1-1`，但用于你反复执行的事情：
- `/weekly-report`
- `/standup-notes`
- `/research-competitor`

将会在未来模块出现。

## 预览：MCP（Model Context Protocol）

把 Claude 连接到外部服务：
- Notion
- Google Calendar
- 数据库
- 各类 API

你需要什么，Claude 就能连什么。

将会在未来模块出现。

## 预览：浏览器自动化

```bash
claude --chrome
```

让 Claude 连接到你的浏览器：
- Claude 可以看到你看到的内容
- 可以操作 Web 应用
- 自动化浏览器工作流

将会在未来模块出现。
