## 1.4：命令与导航

- **预计用时：** 20 分钟

> **互动课程：** 在 Claude Code 里运行 `/start-1-4`

## Slash Commands 速查

### 会话管理
| 命令 | 说明 |
|---------|-------------|
| `/clear` | 开始一个全新对话（保留文件，清空聊天） |
| `/compact` | 压缩对话以节省上下文（适合长会话） |
| `/resume` | 关闭 Claude Code 后继续上次进度 |

### 模型与用量
| 命令 | 说明 |
|---------|-------------|
| `/model` | 切换模型（Sonnet 更快，Opus 更强） |
| `/usage` | 查看配额使用情况 |
| `/cost` | 查看当前会话费用拆分 |

### 上下文与文件
| 命令 | 说明 |
|---------|-------------|
| `/context` | 查看 Claude 当前知道哪些文件 |
| `/add-dir` | 将一个目录加入上下文 |
| `/init` | 为新项目初始化 CLAUDE.md |
| `/memory` | 查看或编辑本项目的记忆 |

### Agents 与权限
| 命令 | 说明 |
|---------|-------------|
| `/agents` | 创建与管理自定义 sub-agent |
| `/permissions` | 查看与管理工具权限 |

### 配置
| 命令 | 说明 |
|---------|-------------|
| `/config` | 打开配置设置 |
| `/doctor` | 运行健康检查与诊断 |
| `/terminal-setup` | 配置终端集成 |
| `/vim` | 切换 vim 按键模式 |

### 其他
| 命令 | 说明 |
|---------|-------------|
| `/help` | 查看所有可用命令 |
| `/bug` | 向 Anthropic 报告 Bug |
| `/quit` / `/exit` | 退出 Claude Code |

输入 `/` 可以打开完整菜单（含描述）。

## 自定义 Slash Commands

- 它们是 `.claude/commands/` 里的文件
- 适合把经常做的事变成快捷方式
- 你也可以创建自己的命令（直接让 Claude 帮你写）
- **重要：** 添加新命令后需要重启 Claude Code

## Escape 中断

按 **Escape** 可以让 Claude 立刻停止输出：
- 当你发现 Claude 方向不对时非常有用
- 控制权永远在你手上

## Rewind 功能（Esc×2）

连续按 **Escape 两次** 可以看到 rewind 选项：
- 只重置对话（保留 Claude 创建的文件）
- 重置对话并撤销文件改动

像 AI 工作的“时间旅行”，非常适合大胆试错。

## Think 关键词

| 关键词 | 效果 |
|---------|--------|
| `think` | 正常深度 |
| `think harder` | 更深分析 |
| `ultrathink` | 最大思考强度（会有彩虹提示） |

适合复杂分析与策略性决策。

> **注意：** 会消耗更多配额。

## 三种输入模式

用 **Shift+Tab** 切换：

| 模式 | 行为 | 适合场景 |
|------|----------|----------|
| **Edit**（默认） | 改动前会请求你授权 | 安全、学习阶段 |
| **Auto-accept** | 直接执行不再反复确认 | 更快、可信工作流 |
| **Plan** | 先给计划，你确认后再执行 | 多步骤复杂任务 |

## “火力全开”模式

```bash
claude --dangerously-skip-permissions
```

以“完全不弹权限确认”的方式启动 Claude。速度最快，但风险也最大：只在你非常信任工作流时使用。

## 快捷键小结

| 快捷键 | 作用 |
|----------|--------|
| `Escape` | 中断 Claude |
| `Escape × 2` | 打开 rewind 选项 |
| `Shift+Tab` | 切换输入模式 |
| `Ctrl+`` | 在编辑器中打开终端 |
| `Ctrl+V` | 粘贴图片（Mac 不是 Cmd+V！） |
