## 1.2：文件探索与可视化工作区

- **预计用时：** 15 分钟

> **互动课程：** 在 Claude Code 里运行 `/start-1-2`

## 双窗格工作流

**问题：** 终端很适合和 Claude 对话，但你看不到文件。

**解决方案：** 在 Claude Code 旁边配一个可视化编辑器。

| 窗格 | 用途 |
|------|------|
| **左侧** | 文件浏览器（看见一切） |
| **底部** | Claude Code 终端（操作一切） |

## 选择你的工具

### Nimbalyst（推荐）

**适合：** 非技术用户，想要最顺滑的 Claude Code 体验

![Nimbalyst - Claude Code 可视化工作区](/images/nimbalyst-hero.webp)

[Nimbalyst](https://nimbalyst.com?ref=carlvellotti&utm_source=carlvellotti&utm_medium=ccforeveryone&utm_campaign=partner) 是一个为 Claude Code 量身打造的免费工作区，特点包括：

- **所见即所得的 Markdown 编辑** - 看格式化后的内容，而不是原始语法
- **文件差异对比（diff）** - 清楚看到 Claude 改了哪里
- **可见隐藏文件夹** - 能看到 `.claude/` 文件夹（有些编辑器看不到）
- **内置 Claude Code** - 在 Nimbalyst 里直接和 Claude 协作

**[下载 Nimbalyst →](https://nimbalyst.com?ref=carlvellotti&utm_source=carlvellotti&utm_medium=ccforeveryone&utm_campaign=partner)**

### Cursor

**适合：** 想把 Cursor 自带的 AI 功能和 Claude Code 一起用的用户

[Cursor](https://cursor.com) 是一个免费编辑器，可与 Claude Code 搭配使用：

- Claude 工作时，你可以在界面里直观看到文件变化
- Cursor 自带 AI 功能，你可以和 Claude Code 配合使用
- 如果你更习惯 VS Code，也完全可以用 VS Code

**[下载 Cursor →](https://cursor.com)**

## 工具对比

| 功能 | Nimbalyst | Cursor |
|---------|-----------|--------|
| **最适合** | 非技术工作 | Cursor AI + Claude Code |
| **Claude Code** | 内置 | 在终端运行 |
| **可见 `.claude/` 文件夹** | 是 | 是 |
| **WYSIWYG Markdown** | 是 | 预览模式 |
| **价格** | 免费 | 免费 |

## 配置指南

### Nimbalyst 配置

1. **[下载 Nimbalyst](https://nimbalyst.com?ref=carlvellotti&utm_source=carlvellotti&utm_medium=ccforeveryone&utm_campaign=partner)** 并安装
2. 打开 Nimbalyst
3. 用 Nimbalyst 打开你的课程文件夹
4. Claude Code 已内置——直接开始！

### Cursor 配置

1. 在 [cursor.com](https://cursor.com) 下载
2. 运行安装程序
3. 打开 Cursor 并打开你的课程文件夹
4. **重要：** 左上角选择 “Editor”（不要选 “Agents”）
5. 打开终端：`Ctrl+``（反引号键，Tab 上方）
6. 输入 `claude` 并回车

## 探索陌生的文件结构

Claude Code 运行后，你可以这样问：

```
Show me the file structure
```

```
Summarize the contents of the docs/ folder
```

```
What files are in this project?
```

Claude 可以帮你快速梳理任何文件夹的结构。

## Markdown 文件

- `.md` 扩展名 = Markdown 格式
- 文档领域的常见标准 - 大模型也很喜欢
- **预览模式：** `Cmd/Ctrl+Shift+V` 查看格式化效果（在 Cursor 中）
- **Nimbalyst：** 默认显示格式化效果

## 真实场景

- **接手同事的项目** - Claude 快速把结构讲清楚
- **客户发来一堆乱七八糟的素材** - 几分钟搞定整理与定位
- **Downloads 文件夹混乱** - Claude 帮你清理归档
- **刚入职** - 快速理解团队共享盘结构
- **任何陌生文件结构**
