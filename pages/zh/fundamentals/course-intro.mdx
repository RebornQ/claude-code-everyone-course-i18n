## 1.1：课程介绍

- **预计用时：** 10 分钟

> **在 Claude Code 里开始本模块：** 运行 `/start-1-1` 开启互动学习。

## ▶️ 如何开始的提醒

⚠️ **如果你还没有下载并解压课程文件：** 请先回到 [模块 0.2：下载课程并开始](/zh/getting-started/start-and-clone)。继续之前，你需要把课程文件放到你的电脑上。

**在 Claude Code 里启动本课程有两种方式：**

### 方式 1：在文件管理器里找到它

进入你的 Documents 目录，找到 `cc4e-course` 文件夹。然后右键并选择 “New Terminal at Folder”（或类似选项）。

大概长这样：

![Opening folder from file explorer](/images/folder-location.png)

然后输入 `claude` 并回车。

### 方式 2：从终端打开

在终端里输入：
```bash
cd ~/Documents/cc4e-course
claude
```

这会进入课程文件夹，并让 Claude Code 直接在其中启动。

**不管用哪种方式，最终都会到同一个地方。然后，开始本模块：**

> **在 Claude Code 里开始本模块：** 运行 `/start-1-1` 开启互动学习。

这就是你启动每一个模块的方式！

## 这门课怎么学？

- **通过动手来学** - 不是刷视频
- **Claude 在终端里直接教你**
- **参考指南**（你正在读的页面）提供更深的解释
- 每一节互动课都有对应的参考页面（ccforeveryone.com）

### 互动课程

每一节课都由一个 slash command 开始：
- `/start-1-1`、`/start-1-2` 等

Claude 会一步步带你做实操练习。你不是“看懂”，而是真做完。

### 参考指南

这些页面提供：
- 更深入的概念解释
- 命令与模式的快速查询
- 更多资源与链接
- 常见问题排查
